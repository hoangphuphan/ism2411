Driving Time Tracker
The programs tracks the user's driving time within a week. With the data collected, the program can estimate the weekly total driving time. Based on the estimate, the user can adjust their driving time accordingly. 

How to run:
Open a terminal in the project folder and run:
python app.py
#or
python3 app.py

Example output:
Welcome to my Python program!
How many minutes did you drive today? 67
Your estimated total driving time this week is 469 minutes
